# Snake Game

A simple browser-based Snake game built with vanilla JavaScript and HTML5 Canvas. Created as a demo to show children how to build simple games.

## How to Play

1. Open `index.html` in a browser.
2. Use the **arrow keys** to control the snake:
   - **←** Move left
   - **→** Move right
   - **↑** Move up
   - **↓** Move down
3. Eat the red food to grow and increase your score.
4. Avoid hitting the walls or your own tail — the game ends on collision.

## Project Structure

| File         | Description                        |
|--------------|------------------------------------|
| `index.html` | Game page with a 400×400 canvas    |
| `snake.js`   | Game logic (movement, food, score) |
| `style.css`  | Layout and styling                 |

## Tech Stack

- HTML5 Canvas
- Vanilla JavaScript (no dependencies)
- CSS
